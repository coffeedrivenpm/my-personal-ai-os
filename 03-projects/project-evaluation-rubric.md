# Project Evaluation Rubric

| Dimension | 1 | 3 | 5 |
|---|---|---|---|
| Problem clarity | Vague | Clear but narrow | Clear, meaningful, user-backed |
| Technical depth | Surface-level | Working basics | Shows tradeoffs and architecture |
| Product thinking | Feature list | Some rationale | Strong user/value framing |
| Evidence | None | Some output | Measurable or demonstrable artifact |
| Communication | Hard to follow | Understandable | Portfolio-ready |
