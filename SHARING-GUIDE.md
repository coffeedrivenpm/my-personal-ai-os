# Sharing Guide

This starter is designed to be shared, forked, and customized.

## Before sharing publicly

- Remove any personal files you added.
- Remove `.git/` if distributing as a ZIP.
- Confirm `private/` contains placeholder files only.
- Confirm `99-archive/` does not contain old personal or work files.
- Search for your name, email, local path, GitHub username, employer names, and secrets.

## Suggested friend handoff message

```text
This is a Claude Code starter vault for personal learning, portfolio work, research, and writing. Start with START-HERE.md and FIRST-RUN-ONBOARDING.md. Do not put sensitive information in the repo unless you understand the privacy model.
```
