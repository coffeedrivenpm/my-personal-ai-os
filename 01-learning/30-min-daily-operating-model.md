# 30-Minute Daily Operating Model

Use this when time is limited.

| Segment | Time | Purpose |
|---|---:|---|
| Recall | 5 min | Review yesterday's concept/output |
| Learn | 10 min | One concept only |
| Build | 10 min | Hands-on task |
| Log | 5 min | Record what changed and next step |

Prompt:

```text
Create a 30-minute learning session for [SKILL]. Include concept, task, output, and self-check. Use my current roadmap and learning style.
```
